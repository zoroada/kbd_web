ZOROKB 固件与文档中心

ZOROKB 是一个使用 C 编写的机械 / 静电容键盘固件，基于 nRF52833 以及性能更强的 nRF54LM20A 驱动，提供低功耗、低延迟、Vial 可配置的键盘使用体验。

多协议连接 USB / BLE / 2.4G 一套固件统一管理
原生 Vial 实时改键、分层、宏与进阶行为支持
面向量产 适合日常使用、售后维护与固件分发
查看 Keyboard 打开使用说明 查看 Vial 教程 下载固件 加入 QQ 交流

核心能力

围绕连接稳定性、可配置性与交付效率构建的功能集。

MX / EC 双路线

同时覆盖机械与静电容方案，固件能力可按硬件配置灵活适配。

原生 Vial

实时修改键位、层、宏与高级行为，不需要重复刷写固件。

BLE 多设备

蓝牙支持 3 台设备切换，适合桌面、平板和移动端混合使用。

2.4G 通道管理

接收器通道与键盘通道联动，适合办公和游戏场景快速切换。

长续航设计

1000mAh 电池下，无灯状态机械约 1000h、静电容约 400h。

拖拽升级

通过 UF2 文件拖拽完成升级，降低售后与维护成本。
