使用说明

蓝牙连接（BLE_1 / BLE_2 / BLE_3）

  1. 在 User 页面找到 BLE_1、BLE_2、BLE_3，设置到键盘按键上。
  2. 按下你要使用的蓝牙通道键。
  3. 在电脑/手机/平板蓝牙页面搜索并连接键盘。
  4. 三个通道可分别绑定三台设备，后续按对应通道键直接切换。

示例：BLE_1=电脑、BLE_2=手机、BLE_3=平板。

需要解绑时，在对应设备内删除该键盘，即可连接新设备。

2.4G 连接（24G_1 / 24G_2 / 24G_3）

  1. 在 User 页面找到 24G_1、24G_2、24G_3 并设置到键盘。
  2. 插入接收器，按下对应的 2.4G 通道键进行切换。
  3. 接收器键值按0/1/2 分别对应 24G_1/24G_2/24G_3。

接收器处于哪个通道，就必须按键盘上对应的 2.4G 通道键，才能正常输入。如果接收器对应的键值>=3,那么有可能会导致无法输入。

灯光功能

  1. 建议把灯光相关按键统一放在 layer1，便于记忆和排错。
  2. 优先设置“灯光开关”和“模式切换”，再设置亮度/速度调节。
  3. 每改一组按键就立刻实机测试一次。

不同固件灯光键名可能不同，以你当前 Vial 键值列表实际显示为准。

休眠与关机（无线模式）

自动/手动休眠

  • 无线模式下（未连接 USB 数据线）无操作约 30 分钟 会自动休眠。
  • 可设置 KB_SLP 按键手动休眠。
  • 休眠后按任意按键即可唤醒。

关机与开机

  • 设置 KB_OFF 后，按下该键进入关机模式。
  • 开机方式 A：插入 USB（电池需接入）。
  • 开机方式 B：长按 Space + S 约 5s。

键盘更新

进入 U 盘模式的 3 种方式

  1. 方法 1：连接 Vial 后，在 Security 页面点击 Reboot to Bootloader。
  2. 方法 2：把 KB_DFU 设置到键盘，按下进入 U 盘模式。
  3. 方法 3：底部复位按钮或复位点快速触发 2 次（间隔需在 0.5s 内）。

刷写步骤

  1. 进入 U 盘模式后，把键盘固件 .uf2 拖入 U 盘（常见盘符 Zorokb_ada）。
  2. 升级过程中不要拔线、不要断电，避免变砖。
  3. U 盘自动消失后等待几秒，键盘会自动重启。

误进 U 盘模式时，可单击复位按钮或短接复位点一次退出。

接收器更新

进入 U 盘模式

  1. 方法 1：Vial 可直接连接接收器，进入 U 盘模式方式同上。
  2. 方法 2：设置 KC_DFU 到键盘，按下触发接收器进入 U 盘模式。

使用 KC_DFU 时，键盘需处于 2.4G 模式，且当前通道与接收器一致。

刷写步骤

  1. 把接收器固件 .uf2 拖入 U 盘。
  2. 升级中不要断电或拔出接收器。
  3. 若出现“报错/无法复制”，通常属正常现象，等待 U 盘自动消失。
  4. U 盘消失后约 3s，拔下并重新插入接收器。

模式切换（U_O_W）

  1. 把 U_O_W 设置到键盘按键。
  2. 按下后可在有线 USB 与无线模式之间切换。
  3. 蓝牙与 2.4G 同时只能有一个模式生效。

键盘会记忆你关机前的模式和通道；休眠/关机后再开机会恢复到上次状态。

电量显示

  • 蓝牙模式下，可在系统蓝牙设备列表查看电量。
  • 有线与 2.4G 下，可在 User 设置PRT_BAT按键后查看。
  • 键盘刚开机时需等待几秒，电量数值才会稳定显示。

重置键盘（KB_RST）

  1. 在 User 页面把 KB_RST 设置到键盘。
  2. 按下后会清空当前配置并自动重启。

重置前建议先导出配置备份，方便后续恢复。

常见问题

  • 蓝牙能搜到但连不上：在对应设备内删除该键盘，即可连接新设备。
  • 2.4G 无输入：检查键盘通道和接收器通道是否一致（24G_1/2/3）。
  • 刷机后异常：重新进入 U 盘模式，再拖入正确 .uf2 文件。
