User Manual

Bluetooth Connection (BLE_1 / BLE_2 / BLE_3)

  1. Open the User page and assign BLE_1, BLE_2, and BLE_3 to keys.
  2. Press the Bluetooth channel key you want to use.
  3. Search for the keyboard in the Bluetooth settings of your computer, phone, or tablet.
  4. Each of the three channels can be paired with one device. After pairing, press the corresponding channel key to switch.

Example: BLE_1 = desktop, BLE_2 = phone, BLE_3 = tablet.

To pair a new device, first remove the keyboard from the old device's Bluetooth list.

2.4G Connection (24G_1 / 24G_2 / 24G_3)

  1. Assign 24G_1, 24G_2, and 24G_3 on the User page.
  2. Plug in the receiver and press the corresponding 2.4G channel key on the keyboard.
  3. The receiver channel is determined by %3: 0 / 1 / 2 correspond to 24G_1 / 24G_2 / 24G_3.

The keyboard and receiver must be on the same 2.4G channel to work.

Lighting

  1. Place lighting-related keys on layer1 for easier memorization and troubleshooting.
  2. Start with power toggle and effect switching, then add brightness and speed controls.
  3. Test the board after every small group of changes.

Actual lighting key names may vary by firmware build. Use the labels shown in your current Vial key list.

Sleep and Power Off (Wireless Mode)

Automatic / Manual Sleep

  • In wireless mode without a USB data cable, the board will automatically sleep after about 30 minutes of inactivity.
  • You can assign KB_SLP to a key for manual sleep.
  • Press any key to wake the board.

Power Off and Power On

  • Assign KB_OFF to a key and press it to power off the keyboard.
  • Power-on method A: plug in USB with the battery connected.
  • Power-on method B: hold Space + S for about 5s.

Keyboard Firmware Update

Three ways to enter USB drive mode

  1. Method 1: connect with Vial, then click Reboot to Bootloader on the Security page.
  2. Method 2: assign KB_DFU to a key and press it.
  3. Method 3: trigger the reset button or reset pads twice quickly within 0.5s.

Flashing steps

  1. Once the board enters USB drive mode, drag the keyboard firmware .uf2 file into the drive, commonly named Zorokb_ada.
  2. Do not unplug the board or cut power during the update.
  3. Wait for the drive to disappear automatically. The keyboard will reboot after a few seconds.

If you entered USB drive mode by mistake, single-click the reset button or short the reset pads once to exit.

Receiver Update

Enter USB drive mode

  1. Method 1: Vial can connect directly to the receiver. Use the same bootloader method as above.
  2. Method 2: assign KC_DFU on the keyboard and press it to trigger the receiver bootloader.

When using KC_DFU, the keyboard must be in 2.4G mode and on the same channel as the receiver.

Flashing steps

  1. Drag the receiver .uf2 file into the mounted USB drive.
  2. Do not unplug the receiver or interrupt power during the update.
  3. If the file copy shows an error, that is usually normal. Wait for the drive to disappear automatically.
  4. About 3s after the drive disappears, unplug and reconnect the receiver.

Mode Switch (U_O_W)

  1. Assign U_O_W to a key.
  2. Press it to switch between wired USB and wireless mode.
  3. Bluetooth and 2.4G cannot be active at the same time.

The keyboard remembers the last active mode and channel before shutdown. Sleep or power cycling restores that state.

Battery Display

  • In Bluetooth mode, you can view battery percentage from the system Bluetooth device list.
  • In wired or 2.4G mode, assign PRT_BAT on the User page to check battery status.
  • After power-on, wait a few seconds for the battery value to stabilize.

Reset Keyboard (KB_RST)

  1. Assign KB_RST on the User page.
  2. Press it to clear the current configuration and reboot the keyboard.

Export a backup before resetting so the configuration can be restored later.

FAQ

  • Bluetooth is visible but cannot connect: remove the keyboard from the old device and pair again.
  • No input in 2.4G mode: verify that the keyboard channel and receiver channel match.
  • Unexpected behavior after flashing: re-enter bootloader mode and drag in the correct .uf2 again.
